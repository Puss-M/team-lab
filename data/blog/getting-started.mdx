# 欢迎加入 Swift Coding Club！🚀

欢迎来到西南财经大学最酷的技术社团！我们是一群对 **AI**、**量化交易** 和 **金融科技** 充满热情的同学。

## 🎯 我们是谁

**Swift Coding Club** 是西南财经大学的 AI 与量化交易社区，成立于 2023 年。我们致力于：

- 💡 **点燃创新思维**：通过技术分享和项目实践，激发创意
- 🤝 **促进协作交流**：连接志同道合的伙伴，共同成长
- 🚀 **实战项目经验**：参与真实的量化策略开发和 AI 应用

## 📅 活动日程

我们每周都有丰富的活动：

### 技术分享会（每周三 19:00）

- **主题**：AI 模型、量化策略、金融科技前沿
- **形式**：成员分享 + 讨论
- **地点**：线上/线下混合

### 项目工作坊（每周六 14:00）

- **内容**：动手实践，完成小项目
- **技能**：Python、机器学习、数据分析
- **成果**：可展示的作品集

### Journal Club（双周日）

- **活动**：阅读和讨论前沿论文
- **领域**：量化金融、深度学习、NLP

## 🎓 如何加入

### Step 1: 填写申请表

访问我们的 [申请页面](/join)，填写基本信息和你的兴趣方向。

### Step 2: 参加面试

我们会安排一次轻松的线上聊天，了解你的背景和期望。

### Step 3: 开始你的旅程

- 加入内部群组
- 获得知识库访问权限
- 分配导师（Mentor）

## 💻 技术栈

在 Swift，你将接触到：

**前端开发**

- React, Next.js, TypeScript
- TailwindCSS, Framer Motion

**后端 & API**

- FastAPI, Django
- PostgreSQL, Redis

**数据科学 & AI**

- Python (NumPy, Pandas, Scikit-learn)
- PyTorch, TensorFlow
- Jupyter Notebook

**量化交易**

- QuantLib, Backtrader
- 向量化回测框架
- Alpha 策略开发

## 🌟 成员收获

加入 Swift，你将：

1. ✅ **技能提升**：从零基础到独立完成项目
2. ✅ **人脉拓展**：认识优秀的同学和业界导师
3. ✅ **作品集**：参与真实项目，丰富简历
4. ✅ **内推机会**：优先获得合作企业实习/工作机会

## 📚 学习路径

我们为不同基础的成员准备了学习路线：

### 🔰 初级路径（0-3 个月）

- Python 基础
- Git & GitHub 使用
- 数据分析入门

### 🎯 中级路径（3-6 个月）

- 机器学习基础
- 量化策略回测
- Web 开发入门

### 🚀 高级路径（6 个月+）

- 深度学习实战
- 生产环境部署
- 独立项目开发
