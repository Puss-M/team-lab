# 新成员入职指南 🎉

恭喜你成为 **Swift Coding Club** 的一员！这份指南将帮助你快速了解社团，融入团队，开启你的技术之旅。

## ✅ 入职清单

### 第一周

- [ ] **完成账号设置**

  - 内部知识库访问权限

- [ ] **了解团队**

  - 了解组织架构
  - 认识导师（Mentor）

- [ ] **环境配置**
  - 安装 Git 并配置 SSH Key
  - 安装 Python 3.11+
  - 安装 VS Code 及插件

### 第二周

- [ ] **项目参与**
  - 选择感兴趣的项目组

### 第三周

- [ ] **社交融入**

  - 参加技术分享会
  - 参加 Journal Club（如有）

- [ ] **技能提升**
  - 根据兴趣方向，制定学习计划
  - 与导师讨论职业规划

## 🛠️ 开发环境配置

### 1. Git 配置

```bash
# 配置用户信息
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# 生成 SSH Key
ssh-keygen -t ed25519 -C "your.email@example.com"

# 添加 SSH Key 到 GitHub
cat ~/.ssh/id_ed25519.pub
# 复制内容，到 GitHub Settings > SSH Keys 添加
```

### 2. Python 环境

我们推荐使用 **pyenv** 管理 Python 版本：

```bash
# Windows (使用 pyenv-win)
pip install pyenv-win --target %USERPROFILE%\.pyenv

# macOS/Linux
curl https://pyenv.run | bash

# 安装 Python 3.11
pyenv install 3.11.7
pyenv global 3.11.7

# 验证
python --version
```

### 3. VS Code 插件

必装插件：

- **Python** (Microsoft)
- **Pylance** (Microsoft)
- **ESLint** (前端必须)
- **GitLens**
- **Prettier**
- **Remote - SSH** (远程开发)

推荐插件：

- **Jupyter**
- **Docker**
- **Thunder Client** (API 测试)

### 4. 项目环境

克隆项目并安装依赖：

```bash
# 克隆仓库
git clone git@github.com:swift-coding-club/alpha-platform.git
cd alpha-platform

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 运行项目
python main.py
```

## 🎯 项目选择指南

根据你的兴趣和技能，选择合适的项目组：

- 提问前先搜索，避免重复问题

❌ **不应该做**：

- 不要在群里发"在吗"就不说事
- 不要截图代码，而是使用代码块
- 不要深夜发工作消息（除非紧急）

## 💡 常见问题 FAQ

### Q: 我完全是新手，能跟上吗？

**A**: 完全可以！我们有完善的新人培养体系，导师会根据你的节奏指导学习。很多优秀成员都是零基础加入的。

### Q: 需要投入多少时间？

**A**: 平均每周 5-10 小时。包括：

- 项目开发：3-5 小时
- 学习时间：2-3 小时
- 活动参与：1-2 小时

学期忙的话可以适当减少，关键是保持连续性。

### Q: 中途可以换项目组吗？

**A**: 当然可以！建议先在一个组深入学习 1-2 个月，再考虑转组。跨组合作也很欢迎。

### Q: 如何获得实习/工作推荐？

**A**: 在社团有突出贡献的成员，我们会优先推荐给合作企业。往届成员已有多人通过社团获得头部券商、量化私募的 offer。

### Q: 毕业后还能参与吗？

**A**: 欢迎往届成员持续参与！可以：

- 远程贡献代码
- 回来做技术分享
- 提供职业指导
- 参与校友网络
