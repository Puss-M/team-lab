## 🌟 关于 Swift Coding Club

### 我们的愿景

**Where Finance Meets Intelligence** — 用 AI 重塑金融

我们致力于培养兼具金融思维和技术能力的复合型人才，通过实战项目和深度学习，打造西财最具影响力的技术社团。

### 成立背景

Swift Coding Club 成立于 2023 年秋季学期，由一群对量化交易和 AI 技术充满热情的同学发起。短短一年多，我们已经：

- 👥 **成员规模**：50+ 核心成员
- 🚀 **项目数量**：完成 12+ 技术项目
- 📚 **技术分享**：举办 30+ 场分享会
- 🏆 **竞赛获奖**：全国数学建模竞赛、量化投资大赛等奖项

### 各部门职责

**技术部门**

- 前端组：网站、Dashboard 开发
- 后端组：API、数据库、DevOps
- 算法组：量化策略、ML 模型

**产品部门**

- 项目管理与规划
- 用户需求分析
- 产品设计与优化

**运营部门**

- 新媒体运营（公众号、B 站）
- 活动策划与执行
- 对外合作与联络

## 💎 核心价值观

### 1. 开放协作 🤝

我们相信知识共享的力量。在 Swift：

- ✅ 所有项目代码开源（内部 GitHub Organization）
- ✅ 鼓励 Code Review 和技术讨论
- ✅ 新人友好，老成员积极指导

### 2. 实战至上 💪

不止于理论学习，更注重动手实践：

- ✅ 每个成员至少参与一个项目
- ✅ 定期 Demo Day 展示成果
- ✅ 真实数据、真实场景、真实问题

### 3. 持续学习 📚

技术日新月异，保持学习热情：

- ✅ 每周技术分享会
- ✅ 论文阅读 Journal Club
- ✅ 内部学习资料库

### 4. 追求卓越 🎯

高标准、严要求：

- ✅ 代码质量：Lint、测试、文档
- ✅ 设计美学：不做"能用就行"的产品
- ✅ 用户体验：细节决定成败
